<ins>Installation</ins>  
1. Install your PostgreSQL18 Server.
2. Create a database on your server.
3. In the codebase go to *src > resources > db.properties*
4. Change the url, user and password to match your server setup.
5. Now you can run the software. It will create all the tables, property types etc. for you.
